Watch Demo Video

🧠 Problem

Sending player position using 3 floats (96 bits) at high frequency increases bandwidth usage.

🚀 Approach

📦 Data Compression

  • Reduce float precision
  • Apply quantization
  • Optional packing into smaller types

📡 Transmission

Simulated locally using function calls (no networking library).

🔄 Reconstruction

Decode compressed values and rebuild approximate position.

🎯 Smooth Movement

Use interpolation (Lerp) to avoid jitter.
